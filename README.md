# Inventory Management System (C# - 2D Array)

A simple console-based Inventory Management System built using C# and 2D arrays.

This project is designed for learning purposes to practice fundamental programming concepts in C#, such as arrays, loops, conditionals, and basic data management.

---

## Features

- Add products  
- Update product price  
- View all products  
- Simple menu-driven interface  
- Basic search by product name  

---

## Concepts Used

- 2D Arrays  
- Loops (for, while)  
- Conditional statements (if, switch)  
- Methods  
- Basic input/output handling  

---

Built as a learning exercise to practice C# fundamentals and strengthen problem-solving skills.
